Night Skybox 01 version 1.0.0
------------------------------------------------------------------

	This package contains hand-painted night skyboxes. There are blue and green tones, moon and without moon for each.

	Features:

		• Hand-painted night skyboxes.
		• Blue and green night tones.
		• 24 High-quiality textures.
		• 24 Mid-quiality textures.
		• 24 Low-quiality textures.
		• Demo scene and scripts.
		• Light & render settings guide.
		• Desktop and Mobile friendly.

		• Support all build player platforms.
		
	Compatible:

		• Unity 4.7.1 or higher.
		• Unity 5.3.4 or higher.
	
	Note:

		• Light and RenderSettings guides are in "Night Skybox 01/Presets" folder.
		• See "Light, Fog and Ambient color references.txt" file for text guide.
		• High-quiality textures are in "Night Skybox 01/Textures/1024x1024" folder.
		• Mid-quiality textures are in "Night Skybox 01/Textures/512x512" folder.
		• Low-quiality textures are in "Night Skybox 01/Textures/256x256" folder.
